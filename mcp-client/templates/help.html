How to use the OCL MCP client safely.

Chat Config Help

Quick start

1. Open Chat and enter your provider credentials and OCL token.
2. Adjust the tool list on the Config page if you need to limit access.
3. Ask about repositories, concepts, mappings, or expansions.

Security guidance

Provider keys and OCL tokens are stored only in sessionStorage and are cleared when the tab or browser closes. We do not store any keys on our servers. You remain responsible for your credentials. Disable or rotate keys after you finish to reduce risk.

Need a clean reset?

Clear localStorage to reset tool preferences. Clear sessionStorage or close the browser to remove credentials.
